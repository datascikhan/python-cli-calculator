# 🧮 Python CLI Calculator

A simple and beginner-friendly Command Line Calculator built in Python.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* ❌ Error Handling (divide by zero, invalid input)
* 🔁 Continuous loop until exit

---

## 💻 Technologies Used

* Python 3

---

## 📦 How to Run

1. Clone the repository:

```
git clone https://github.com/datascikhan/python-cli-calculator.git
```

2. Open the folder:

```
cd python-cli-calculator
```

3. Run the program:

```
python calculator.py
```
